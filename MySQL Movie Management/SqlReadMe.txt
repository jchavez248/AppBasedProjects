To the Data Movie Data Management Program follow these steps:

1. Open MySql application and start a local instance of an sql server from your computer
2. Have ready a general schema template
3. Import the code onto a new sql script within your general schema template
4. Save the code and then compile/run the code to create all the needed tables for this project
5. Highlight specific sub query code and run the code to return the expected results for the data

Purpose: The purpose of this project is to create a database for Movies and be able to represent 
         management of this data through various sub queries. Entities within this project include: 
         Actors, Producers, Editors, Directors and Movie.
